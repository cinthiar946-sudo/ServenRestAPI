## O que é :
Este repositório foi criado para o bootcamp de teste de API Rest

## Tecnologia ultilizada
-Postman versão web
-node version v18.6
-newman vs 3.2
-newman-reporter-html

## Documentacões
- Doc da API: [Consulte Swagger] https://serverest.dev/swagger.json

## Como instalar o ambiente
- Primeiro: baixe o node no seu computador : (https://nodejs.org/en/download)
- Segundo: Realize a instalacão do newman de forma global: (https://www.npmjs.com/package/newman-reporter-htmlextra)
- Terceiro: Realize as instalacões da dependencia dos relatórios (Opcional)

## Como rodar os testes

### Pelo postman web ou desktop
- Import a collections e environments
- Execute os testes de forma manual ou automatizada

### newman
- Abra seu console de preferência
- Execute a segunda linha de comando para rodar os testes
